你已经到了<<The Bevy Book>>的结尾了! 不幸的是, 我们甚至没有接触Bevy基础的功能! 最终, 这本书将会涵盖Bevy的各个方面, 但是在此之前, 我们建议你看看:

* **[The Bevy Examples](https://github.com/bevyengine/bevy/tree/master/examples)**: 我们为每一个主要的Bevy功能创建一个示例. 这是目前学习Bevy功能以及如何使用它们的最佳方法. 
* **[Breakout](https://github.com/bevyengine/bevy/blob/master/examples/game/breakout.rs)**: 一个小游戏, 展示了真的Bevy Game应该是什么样子.
* **[Bevy API Docs](https://docs.rs/bevy)**: Bevy的Rust API文档.
